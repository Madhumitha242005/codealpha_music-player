🎵 Modern Music Player Web App

A modern and fully responsive music player web application built using HTML, CSS, and JavaScript.
It features a sleek UI, smooth playback controls, and an interactive playlist experience inspired by contemporary streaming platforms.

✨ Features

🎶 Play, Pause, Next, and Previous controls

🔀 Shuffle and 🔁 Repeat functionality

🎚 Volume control slider

⏱ Real-time progress bar with current time & duration display

📜 Interactive playlist selection

💿 Dynamic album cover display

⌨️ Keyboard media control support

📱 Fully responsive design for mobile and desktop

⚡ Auto-play next track

🛠️ Technologies Used

HTML5 – Structure and Audio API

CSS3 (Flexbox, Glassmorphism, Gradients, Animations) – Modern UI Styling

JavaScript (DOM Manipulation, Event Listeners, Audio API) – Playback Logic & Interactivity
